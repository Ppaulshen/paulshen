This is a book store application that written by java and used mySQL database
to run this program, you need to follow these steps
	1. set mySQL environment
	2. connect to your database
	3. open your editor and run /src/main/BookMain
	4. register a account
	5. log into your account
	6. you can buy books and track the packges by track number
	7. log in as admin(user: admin/password:admin)
	8. allow to manage your sotre
	9. allow to add oe delete books and view the bill
